Coffee Lover Project

Welcome to your Boostrap Project:Coffee Lover

You will recreate this webpage and additional Form Page

Just follow the instruction steps here to re-create this webpage!

  1. Create a general landing page with JumboTron
  2. Next create a NavBar on the page that links to another html file
  3. Add two paragraphs of lorem-ipsum below the jumbotron
  4. Next use the Grid system to add in thumnail pictures of coffee. Source: https://stocksnap.io/search/coffee you will need to use the thumbnail class for this. Here are the image links:
    • One
    • Two

  5. Next make sure that at the largest settings you have 3 colomns. of pictures. on the smallest you should have 3 colomns. its up to you where you want to change
  6. Next create another html file for the Sign Up page.
  7. Create a Form on the Sign Up page with Email, Password and a Check Box

That's it. Fell free to play arround with this project and style it more to your liking. The main focuses of this project are the uses of NavBar, JumboTron, Form, Container, and the Grid System
